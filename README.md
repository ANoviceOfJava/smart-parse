# smart-parse
智能识别地址
例如:小埋13000000000广东省广州市天河区体育西路xx号

一键解析地址为：
收货人：小埋
电话：1300000000
省市区：广东省广州市天河区
地址：体育西路xx


address-parse.js : 智能解析地址js
area-list.js : 基于weui的cityPicker地址选择器组件地址库 

如果地址库不匹配只需要更新area-list 即可


原作者rep地址：https://github.com/wzc570738205/smart_parse
